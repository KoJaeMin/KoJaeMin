## Intro


안녕하세요 **고재민**입니다.

**웹 서버 프로그래밍**에 관심이 있으며 **백엔드 개발자**가 되고 싶은 학생입니다.

평소 최신 기술에 관심이 있으며 기술 개발 블로그 및 공식 문서를 읽고 따라해 보는 것을 좋아합니다.

꾸준히 학습하고 노력하여 더 나은 개발자로 성장하려합니다.
